# Maluma Memes - Plataforma de Fans

**Una plataforma moderna y divertida para la comunidad de fans de Maluma, donde el entretenimiento y los memes se unen para celebrar al Pretty Boy.**

---

## Descripción del Proyecto

**Maluma Memes** es una aplicación web de una sola página (SPA) diseñada para proporcionar entretenimiento y crear una comunidad vibrante alrededor de los memes y la cultura de Maluma. La plataforma ofrece una experiencia interactiva donde los fans pueden disfrutar de contenido dinámico, frases icónicas del artista y conectarse con otros seguidores.

### Necesidad que Resuelve

- **Entretenimiento**: Proporciona una fuente constante de contenido divertido y relevante para los fans
- **Comunidad**: Crea un espacio digital donde los seguidores pueden compartir y disfrutar de memes
- **Interactividad**: Ofrece una experiencia dinámica con contenido que cambia constantemente
- **Accesibilidad**: Plataforma web moderna accesible desde cualquier dispositivo

## Características Principales

### Componentes Dinámicos

1. **Header.jsx**
   - Logo y navegación estilizada
   - Diseño responsive con menú adaptable
   - Efectos visuales modernos con gradientes

2. **HeroMaluma.jsx**
   - Estado de ánimo dinámico de Maluma
   - 10 frases icónicas aleatorias
   - Botón interactivo para cambiar el estado
   - Estadísticas visuales del artista
   - Imagen placeholder con efectos hover

3. **Footer.jsx**
   - Información de créditos completos
   - Enlaces a redes sociales oficiales
   - Diseño en columnas responsive
   - Información legal y derechos de autor

### Características Técnicas

- **Diseño Responsive**: Adaptable a todos los dispositivos
- **Gradientes Modernos**: Estilos visuales atractivos
- **Animaciones CSS**: Transiciones suaves y microinteracciones
- **Componentes Modulares**: Arquitectura escalable y mantenible
- **SEO Optimizado**: Meta etiquetas y estructura semántica

## Instalación y Ejecución

### Prerrequisitos
- Node.js (versión 16 o superior)
- NPM (generalmente incluido con Node.js)

### Pasos de Instalación

1. **Clonar o descargar el proyecto**
   ```bash
   # Si tienes el código fuente, navega al directorio
   cd "c:\Users\PC\Downloads\Proyecto Katherine Daniela Quintero Paez"
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Iniciar servidor de desarrollo**
   ```bash
   npm run dev
   ```

4. **Abrir en el navegador**
   - La aplicación estará disponible en `http://localhost:5173`
